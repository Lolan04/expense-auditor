# 💸 ExpenseAI — Policy-First Expense Auditor

> AI-powered expense auditing system built for FinTech compliance

🌐 **Live Demo:** https://expenseai-frontend-qv8b.onrender.com

---

## 🚀 Features

- 📤 **Employee Portal** — Upload receipts with business justification
- 🤖 **AI Policy Audit** — Auto cross-references 40-page company policy
- 🚦 **Traffic Light System** — Approved / Flagged / Rejected
- 📊 **Finance Dashboard** — Review and override AI decisions
- 📧 **Email Notifications** — Auto alerts on claim status
- 📱 **Mobile Friendly** — Works on all screen sizes

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React.js |
| Backend | FastAPI (Python) |
| OCR | Tesseract |
| AI Audit | LLM + RAG |
| Database | SQLite |
| Hosting | Render |

## ⚙️ Run Locally

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8001
```

### Frontend
```bash
cd frontend
npm install
npm start
```
